### src overwiew
pure source code for project, has no imports 

### Project structure
- `src/`: pure source code
- `src/deps/`: dependencies of project (they are suposed to be injected from wrappers or user)
- `src/deps/fdeclare.<name>.h`: function prototypes (they are atumaticly generated by the wrapper build [wrappers.lua](/builds/wrappers.lua)) using the wrappers on wrappers dir 
- `src/implementation/`: function implementations  
- `src/implementation/PROJECT_NAME_<function_name>.c` : function definitions 
- `src/implementation/imports.fdeclare.h`: function declarations geneerated by silver chain (dont touch it)
- `src/prototype/`: headders prototypes 


### Always Read 
allways read the following files if you need to make a src edition  
-  `src/prototype/*` read all the files 
-  `src/deps/*` read all the files 


### How to edit a function 
1. go to to the `src/prototype/`and edit the function prototype (if need)
2. go to `src/implementation/PROJECT_NAME_<function_name>.c` and edit the function definition
